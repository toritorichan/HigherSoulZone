隱藏彩蛋選單 🥚

除了保留原有的「亂碼變文字」和「愛心破碎」，還想加哪些？可多選！

A

👽 Konami Code 召喚外星人

輸入經典秘技 ↑↑↓↓←→←→BA，整個畫面出現 UFO 降落動畫，背景變成外太空星圖，外星人從飛碟走出來跟你打招呼，持續 5 秒後消失。

B

🌀 Matrix Rain 模式

快速連點頁面 5 下，全頁面觸發《駭客任務》綠色字元雨瀑布效果，所有文字暫時變成亂碼再還原，像是被駭入了一樣。

C

🔮 滑鼠軌跡粒子

滑鼠移動時拖曳出星光粒子尾巴，在外星種類頁面，粒子會變成對應星座的顏色。長按不動 3 秒會聚集成一個小型黑洞漩渦。

D

🎵 隱藏音效互動

點選不同外星人種類時播放不同的科幻音效。在 deadSite 頁面爆炸時有音效搭配。用 Web Audio API 合成，不需額外音檔。

E

👁️ 第三眼開啟

在作者頁面，對「第三眼」相關文字快速點 3 下，整個頁面進入「第三眼視角」：色調反轉為負片效果，隱藏文字浮現（例如座標、神秘訊息），頁面微微扭曲晃動。

F

🛸 滾動觸發 UFO

當使用者滾動到頁面特定位置，一架小型 UFO 會從畫面邊緣飛過，帶著光束掃過內容區域。每次出現的位置和路徑都不同，營造隨機驚喜感。

G

⌨️ 外星語翻譯器

在任何頁面按下 Alt+A，所有中文文字會「翻譯」成外星符號字體（用 CSS 自訂字型或 Unicode 特殊字元），再按一次還原。像是切換了語言模式。
