外星人（英語：extraterrestrial intelligence，簡稱ET），又稱宇宙人，是人類對地球以外的智慧生命的統稱。
從古至今，人類對其他行星是否存在生命，以及外星人及其文明是什麼樣的，一直存在猜測、研究與探索。
一些人將地球上難以用科學解釋的文明產物或不明飛行物，都歸責於外星人及其影響。

有些學術單位（其中較著名的包括哈佛大學、柏克萊加州大學和非營利組織SETI協會等）參與了搜尋地外文明計劃（SETI）。
這些組織致力於用無線電望遠鏡等先進設備接收從宇宙中傳來的電磁波，從中分析有規律的信號，希望藉此發現外星生命。


昴宿星人
/ Pleiadian 		七姐妹星人，是居住在梅西爾星雲星團表編號M45上的昴宿星團，來自其他維度的外星人類。從上古時代的所有古文明國就出現了對昴宿星人的記載，包括澳洲的土著、毛利人、中國和馬雅人，近年在國外因被拍攝到疑似來自昴宿星的UFO飛船而再度活躍。他們不論精神方面及科技方面都要比我們地球人高很多。跟我們地球人一樣，他們也屬於人型外星人，也有男女之分。他們的壽命比我們人類要長，據說可以活到一千年（這裡是換成我們地球的時間）。
天琴星人
/ Lyran 		魔幻小說中地球人以及人型外星人的起源。地球上的白種人高大，皮膚頭髮眼睛的顏色都很淡跟天琴星人很像。當天琴星發生戰爭時，許多人為了躲避戰爭移民到昴宿星，昴宿星及織女星上。約五萬多年天琴星又發生戰爭，當時的領袖及其他三十六萬人來地球避難，成為天琴星最早來到地球的時間點。
灰人
/ Greys 		在外星人目擊報告中占比例很大的一種外星人，也是大眾文化中被人類所熟知的外星人。灰人通常被描述為擁有灰色瘦弱的身體和一雙深邃的黑色眼睛。體型小，被新創造類似蜥蜴人的種族。身高3.5-4.5英尺（一英尺約合0.3米）。皮膚顏色有白色，灰色，白灰色，灰藍色，灰綠色。無性別區分，沒有生殖器。殘忍沒有同情心，他們是最常見的外星人種類。他們跟美國有秘密合作的關係，美國得到飛碟及其他武器的技術，GERYS得到資源&能源&複製GREYS的基地，在地球上，他們通過基地內的實驗室培育其他的GREYS。
爬蟲人
/ Reptilians
外星爬蟲人時常與外星人誘拐事件還有陰謀論相提並論，亦有人認為自古以來的文明就是爬蟲人所創造，甚至還認為他們是人類的祖先。這些爬蟲人的外表有淺綠色的麟片覆蓋，有琥珀色的眼睛。 手指尖長，牙齒尖，五至六米或15呎高，懂得心靈感應。傳統上相信它們能夠變形，有食用人類或吸飲人血的習慣，以保持人類的外型。陰謀論者還相信現時世界有不少政要其實也是它們的同類變形而成的。
獵戶星人
/ Orions
獵戶星人有75%的人型外星人，14%的爬蟲類外星人，10%黑皮膚天琴星人及1%白皮膚天琴星人。大部分的銀河戰爭都跟獵戶星人有關。 位於銀河系的某處，獵戶座是個無與倫比的輝煌的原始天堂世界，它能夠以某種方式改變它的生態，從它的地質、大氣和重力，甚至整個生態系統的生物化學，成為任何殖民它的物種最適合居住的星球。這個星球擁有財富、資源和超先進技術的寶庫。他們的文明將比其他任何人都繁榮，他們的種族將被推進到遠古種族的狀態，與古代傳說中的安塔拉人平起平坐。
然而，從來沒有人登陸過這個星球，或查明這些故事是否屬實，因為遠古獵戶座族在很久以前就拋棄了這個星球，他們留下了守護者。守護者是一個大型自動移動戰鬥站，擁有先進的武器和防禦系統，可以保護地球免受任何敢於宣稱它的人的侵害。
天龍星人
/ Draco 		爬蟲類外星人。屬於「皇室統治階層」。他們外觀與其他外星人不同，天龍星人有翅膀，一般而言他們有7-12英尺高。而且他們層級制度是依照他們皮膚顏色來區分的，白色皮膚的天龍星人是最高級的。 據說天龍星人精通遺傳學，且他們存在這個宇宙數十億年了，他們也創造了數個種族，也因為這樣他們天龍星人也想當這些「被造物」的神，面對這些種族，他們天龍星人有絕對的優越感。 他們在銀河系大部分可被發現，而且他們也攻取獵戶座大部分的地區。或許他們會取得所有獵戶座的控制權。
天狼星人
/ Sirians 		又叫多貢人，是非洲西部馬利共和國境內的一個原始的部族。他們的生產力非常落後，還過著很原始的生活。然而，令人驚異的是，許久以來，他們就一直崇拜著一顆遙遠的、用肉眼無法看到的星星——天狼伴星。更奇怪的是，他們還知道這顆天狼伴星具有橢圓形的軌道；知道它圍繞天狼星運行的周期：甚至還知道它是一顆體積小、密度很大的天體。顯然，這些知識與他們所具有的其他知識水平相比，是非常不相稱的。
天蛾人
/ Mothmen 		是一種不明的奇異生物，可以隱形，此時只有同伴相互可見，雌體較雄體強壯，每一具雌體都是雄體的啟蒙者。大多數看到天蛾人的人都將其描述成有翅膀的人形生物，有翼，有一對可怕的折射紅色光芒的眼睛。它好像沒有頭，眼睛長在胸這裡。他們生活在地下，有著像蝙蝠般的大翅膀，也擁有角。根據遇過他們的人描述，他們很聰明也很有惡意。天蛾人除了嚇人之外似乎很少有做過什麼壞事的傳說。他們在西維吉尼亞州及新墨西哥州被遇見過。
